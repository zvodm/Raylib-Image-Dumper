# Raylib Image Dumper

A small command-line tool that converts image files into C++ header files using raylib's `ExportImageAsCode()` function.

## Features

* Loads images supported by raylib.
* Exports each image as a standalone C++ header.
* Supports converting multiple images in a single command.
* Keeps the original filename and replaces the extension with `.hpp`.

## Building

This project requires:

* C++
* raylib

Compile it with your preferred compiler and link against raylib.

## Usage

```bash
rid <image file(s)>
```

Example:

```bash
rid image.png logo.jpg sprite.bmp
```

This generates:

```
image.hpp
logo.hpp
sprite.hpp
```

## Output

Each input image is converted into a C++ header containing the image data generated by raylib's `ExportImageAsCode()` function.

The generated headers can be included directly in a raylib project, allowing images to be embedded into your executable instead of being loaded from external files.

## Example

```cpp
#include "image.hpp"

// Use the generated image data with raylib.
```
